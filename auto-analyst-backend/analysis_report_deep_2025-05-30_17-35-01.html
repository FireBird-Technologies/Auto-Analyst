Key Factors Influencing Property Prices

Statistical Relationships

Correlation analysis revealed several strong relationships between property characteristics and price: - Property area showed the strongest correlation with price (0.55) - Bathrooms (0.51) and total rooms (0.52) also demonstrated strong positive correlations - Room density showed a negative correlation (-0.26), suggesting that cramped properties command lower prices

ANOVA tests identified statistically significant price differences based on: - Mainroad access (F=57.34, p<0.0001) - indicating a substantial premium for properties with main road access - Basement (F=6.17, p=0.0133) - properties with basements showed significantly different pricing

Interestingly, some features that might intuitively seem important showed no statistically significant impact on price: - Guestroom (p=0.9619) - Hot water heating (p=0.5171) - Air conditioning (p=0.5828)

Feature Importance from Machine Learning

The machine learning model (Random Forest Regressor) achieved an R² score of 0.5689, explaining approximately 57% of the variance in property prices. The model identified the following as the most important price determinants:

  1. Property area (51.9% importance)
  2. Number of bathrooms (14.6%)
  3. Number of stories (6.8%)
  4. Parking availability (5.8%)
  5. Number of bedrooms (5.2%)

The remaining features (air conditioning, guestroom, basement, furnishing status, and preferred area) each contributed less than 2.5% to price predictions.

Property Market Segmentation

Premium Features Analysis

Based on the comprehensive analysis, several key insights emerge:
